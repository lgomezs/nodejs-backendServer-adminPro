BACKEND NODEJS con expressjs, ONEXION A MONGO.

	Base de datos (crear cuenta en mlab o usar su propia BD local)
		Para la BD, se crearon los collection en https://mlab.com/
		
	Librerias instaladas:
	
		--instal mongoose
			npm install mongoose
		--install nodemoon
				npm install -g nodemon
				
		--para tratar body en el request
		 npm install body-parser --save
		 
		--mongoose-unique-validator
			npm install --save mongoose-unique-validator
			
		--bcrypt.js	 (para ecncriptar claves)
			npm install bcryptjs --save
			
		--node-jsonwebtoken
			npm install jsonwebtoken --save
			
		--express-fileupload: Para trabjar con carga de imaganes
			express-fileupload
			npm install --save express-fileupload --save
		
		--install google auth
			npm install google-auth-library@0.12.0 --save
			
			
	Iniciar el proyecto:
		npm start
